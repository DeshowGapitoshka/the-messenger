# 0.0.1

Server:

Web-сервер имеет два поля.

/health - состояние Web-сервера
/messages - JSON-объекты {"Data": сообщение, "Id": айди пользователя}

Имеется обработка ошибок, получение POST и GET запросов.

Client:

Отправка POST-запросов
У пользователя запрашивается сообщение. Поле Id статично: 0




# 0.0.2

Server:

Теперь данные о сообщении хранятся на БД, а не в коде сервера.
Добавлено поле ID - порядковый номер в БД.

Client:

Без изменений

Database:

Реализована при помощи PostgreSQL.
Содержит поля:
ID - Порядковый номер в БД.
User_id - Идентификационный номер клиента.
Message - Сообщение отправленное клиентом.

# 0.0.3
